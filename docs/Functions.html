  • Log(message:_:_:_:)

    Helper log method for more accurate console prints.

    Print will only happen on DEBUG.

    Inspired by https://github.com/JungleCandy/LoggingPrint/blob/master/LoggingPrint.swift

    Declaration

    Swift

    public func Log (message: String,
                 _ file: String = #file,
                 _ function: String = #function,
                 _ line: Int = #line)

    Parameters

    object

    Object to be printed. Can be an expression.
    file

    Name of the file calling this function. By default, the source file without .swift extension.
    function

    Name of the function calling this function. By default, same name as function where this is called.
    line

    Line number where this function is called. By default, line number within the file where the call is made.
  • Log(_:log:type:_:_:_:)

    Helper log method for more accurate console prints.

    Prints using OSLog. The logAccess set the visibility of the logs. If it is private the log message doesn’t appear in the logs console when the device isn’t connected.

    https://developer.apple.com/documentation/os/logging

    Declaration

    Swift

    @available(OSX 10.12, iOS 10.0, watchOS 3.0, tvOS 10.0, *)
public func Log(_ message: String,
                log: OSLog = OSLog.default,
                type: OSLogType = OSLogType.default,
                _ logAccess: LogAccess = .public,
                _ function: StaticString = #function,
                _ line: Int = #line)

    Parameters

    message

    Object to be printed. Can be an expression.
    log

    OSLog object for this log.
    type

    OSLogType for this log.
    logAccess

    the level of privacity of the log. By default is public.
    function

    Name of the function calling this function. By default, same name as function where this is called.
    line

    Line number where this function is called. By default, line number within the file where the call is made.
